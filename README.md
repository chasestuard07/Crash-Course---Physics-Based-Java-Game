# Crash-Course - Physics Based Java Game
A small, physics-based game coded in raw java. 


## Mechanics   
Control 2-5 orbiting planets in an attempt to avoid meteors.  
Each planet starts with 3 health; grey meteors remove one and blue meteors add one.  
Score is based on how many meteors in total are destroyed.   

## Controls
Buttons: left-click  
Moving Planets: left-clicking and dragging   

That's all!!
